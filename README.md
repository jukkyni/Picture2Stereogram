# Picture2Stereogram
一枚の写真から,平行法または交差法によって裸眼立体視できるステレオグラムを生成します.

## 開発環境
device:MacBook Air M1
os:macOS sonoma 14.6.1
python:3.12.3
use packages:
    opencv-python=4.10.0
    torch=2.4.0
    torchvision=0.19.0
    numpy=1.26.4
    scikit-learn=1.5.1
    matplotlib=3.9.1.post1
Depth-Anything-V2
